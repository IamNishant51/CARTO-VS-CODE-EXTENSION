# Changelog

All notable changes to the **Carto** extension will be documented in this file.

## [1.0.1] - 2026-05-08

### Added
- **Multi-Provider AI**: Integrated support for Google Gemini, OpenAI, Groq, and local Ollama models.
- **Glassmorphism UI**: A modern, premium user interface with translucent surfaces and animated backgrounds.
- **Intelligent Bundling**: High-performance context gathering with automatic tech stack detection.
- **Security Scanning**: Automated detection and exclusion of sensitive files (API keys, credentials, etc.).
- **Live Preview**: Real-time rendered Markdown preview of the bundled context.
- **Secure Configuration**: Persistent, local-only storage for API keys and provider preferences.

### Changed
- Refined Activity Bar icons for better theme compatibility.
- Switched all branding assets to high-resolution SVGs.
- Improved context truncation logic to respect provider-specific token limits.

### Fixed
- Resolved UI state synchronization issues between the extension and webview.
- Fixed sidebar icon rendering as a solid block by implementing a dedicated mask icon.
